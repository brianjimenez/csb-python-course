1. NumPy

What is NumPy?

"NumPy is a library for the Python programming language, adding support for large, multi-dimensional arrays and matrices, along with a large collection of high-level mathematical functions to operate on these arrays." - Wikipedia

Arrays

Arrays are the simplest object in NumPy (a multidimensional list) from the same data type. 


import numpy as np

p = np.array([1, 2, 3, 4, 5])
q = np.array([1, 2, 3, 4, 5], dtype=np.float)
>>> p
array([1, 2, 3, 4, 5])
>>> q
array([1., 2., 3., 4., 5.])

Arrays

In NumPy, array dimensions are called axis and the total number of axes, rank.

array is an alias of the numpy.ndarray object.

  • ndarray.ndim: #axis of the array object (matrix)
  • ndarray.shape: Tuple of dimensions of the array
  • ndarray.size: Total number of elements

Example:


p = np.array([1, 2, 3, 4, 5])
>>> p
array([1, 2, 3, 4, 5])
>>> p.ndim
1
>>> p.shape
(5,)
>>> p.size
5

Matrix operations


a = np.array([[1,0], [0,1]])
b = np.array([[1,2], [3,4]])
>>> a+b
array([[2, 2],
       [3, 5]])
>>> a*b
array([[1, 0],
       [0, 4]])
>>> a.dot(b)
array([[1, 2],
       [3, 4]])
>>> a**2
array([[1, 0],
       [0, 1]])

2. Matplotlib

Library structure

Matplotlib code is divided in three different parts:

  • backend: the system dependent part (file outputs, window manager, etc.)
  • Maplotlib API: base code
  • pyplot: procedural interface similar to Matlab and OpenGL

pyplot

pyplot relies on global state and a flat namespace.

It is OK for small applications, but when complexity raises, OO code is preferred.

Simple example


import matplotlib.pyplot as plt
import numpy as np

x = np.arange(0, 10, 0.2)
y = np.sin(x)
fig = plt.figure()
ax = fig.add_subplot(111)
ax.plot(x, y)
plt.show()
